CORE COMMERCE

Payment Gateway

Razorpay integration covering UPI, credit/debit cards, net banking and wallets. Payment verification, order confirmation, invoice generation and notification on success.

INTEGRATION
📦

Order Management

Five-stage status pipeline: Placed → Processing → Dispatched → Out for Delivery → Delivered. Real-time push + WhatsApp notifications at each stage change.

Risk Analysis

Risk Severity Mitigation
Referral self-abuse / duplicate rewards HIGH Validate unique IP + device, block self-referral at backend, admin approval workflow for large commissions.
Payment failure / webhook mismatch HIGH Idempotent payment verification, Razorpay webhook signature validation, retry queue via Celery.
WhatsApp API rate limits MED Message queue with Celery + Redis, exponential backoff on 429 errors.
GST invoice compliance errors MED Validate GST% per product category, test invoice format against GST portal requirements before launch.
Flutter + FastAPI version drift MED Pin SDK/library versions. Repository pattern ensures frontend changes don't affect backend and vice versa.
AWS cost overrun LOW Use RDS t3.micro for dev, auto-scaling ECS for prod. CloudFront CDN reduces S3 bandwidth costs.
OTP delivery failure (email) LOW Fallback to SMS OTP. Rate-limit resend requests to prevent spam.
